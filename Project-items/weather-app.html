The Personal Dashboard is a minimalist, real-time browser homepage that combines useful daily info like live time, weather conditions, and crypto prices—all displayed over a dynamic background. Built as part of the Scrimba Frontend Developer Career Path, this project focuses on working with multiple APIs, asynchronous JavaScript, and user-centered layout design.

I used the Unsplash API for rotating background images, the CoinGecko API for cryptocurrency data, and OpenWeatherMap for real-time weather updates based on geolocation. JavaScript’s fetch, async/await, and try/catch were used extensively to handle requests gracefully and keep the interface responsive even with unreliable data sources.

Through this project, I became more confident with async programming, working with external APIs, and managing dynamic DOM updates. It also helped me design a simple but effective user interface using semantic HTML and custom CSS. The result is a fully functional, visually appealing personal homepage that delivers value at a glance.
