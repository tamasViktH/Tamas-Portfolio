Van Life Rental Platform Demo

HTML & CSS

GitHub Repo Live version

Van Life is a fully responsive web application built during Scrimba’s Frontend Developer Career Path. It mimics a real-world rental platform where users can browse camper vans, view detailed listings, and log in to access a personalized host dashboard. This project showcases dynamic routing, data fetching, and user authentication within a React environment.

The app was developed using React, React Router v6, and styled with custom CSS. It incorporates a combination of MirageJS and Firebase Firestore to simulate and retrieve real-time backend data. Key features include protected routes with an AuthRequired component, nested layouts using React Router’s `Outlet`, filterable van listings, and mock login logic with persistent state.

Working on Van Life significantly deepened my understanding of modern routing techniques, context-based layouts, route protection, and external API handling. It also helped me build experience with scalable UI architecture and responsive mobile-first design. This project stands out as a comprehensive demonstration of my ability to structure and implement a realistic single-page application from scratch.
