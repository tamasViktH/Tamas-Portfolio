Assembly Gameweb app

HTML & CSS

GitHub Repo Live version

The Assembly Game is an interactive typing challenge created as part of the Scrimba Frontend Developer Career Path. The goal is to quickly and accurately type randomly selected words before they vanish, making it a fun way to practice both coding and keyboard skills.

This project was built with React, using modern JavaScript to manage state, handle user input, and update the UI in real-time. I implemented a custom random word generator, managed dynamic rendering, and ensured a responsive layout using CSS.

Through this project, I deepened my understanding of React components, hooks, event handling, and how to build interactive user experiences. It also gave me valuable practice in breaking down a UI into reusable parts and designing intuitive game mechanics.
