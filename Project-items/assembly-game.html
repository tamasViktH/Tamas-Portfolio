GitHub Repo Live version

Assembly Endgame is an interactive typing game built as part of the Scrimba Frontend Developer Career Path. The player must guess the names of programming languages letter by letter before running out of lives, making it a fun and challenging way to practice both coding logic and typing accuracy.

This project was developed using React with modern JavaScript. I managed real-time input handling, dynamic UI updates, and interactive feedback using state hooks and conditional rendering. A custom word randomizer and a responsive virtual keyboard bring the game to life across devices.

Through this project, I expanded my skills in React components, hooks, event-driven logic, and UI/UX design. It helped me deepen my understanding of how to structure a game interface with reusable elements, create a smooth user experience, and handle live gameplay logic in a clean and scalable way.
