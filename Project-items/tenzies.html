Tenziesweb app

HTML & CSS

GitHub Repo Live version

Tenzies is a fast-paced, interactive dice game I built as part of the Scrimba Frontend Developer Career Path. The goal is to roll ten dice until they all show the same number, with the added challenge of “freezing” specific dice between rolls to hold their values. The game offers a visually engaging experience, complete with responsive design and celebratory confetti effects when the player wins.

This project was developed using React and showcases several core concepts such as state management with `useState`, side effects with `useEffect`, and component reuse. Each die is rendered as a self-contained component that tracks its held status and updates in real time. I also explored keyboard accessibility using `useRef`, ensuring a smooth experience for all users.

Building Tenzies helped me deepen my understanding of user interaction, game logic flow, and creating responsive, polished interfaces. I used tools like `nanoid` for unique IDs and styled everything with semantic HTML and custom CSS. This project gave me hands-on experience translating an idea into a complete, functional mini-app — and reinforced my confidence in building interactive components from scratch.
