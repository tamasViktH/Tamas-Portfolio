Chef Claude AI Recipe Assistant

React | JavaScript | Vite | CSS | Anthropic API

GitHub Repo Live version

Chef Claude is a recipe-generating AI assistant built with React, featuring real-time integration with Anthropic's Claude 3 model. Originally developed as part of the Scrimba Frontend Developer Career Path, I later remodeled and expanded the project to improve both its functionality and design.

The app helps users generate meal ideas based on ingredients they already have. Users can input one or more ingredients, manage their list, and receive AI-generated recipe suggestions presented in a clean, Markdown-rendered format.

I implemented this using modern React patterns—including hooks, controlled inputs, and asynchronous API requests. All communication with Claude AI is securely handled via environment variables, ensuring that no keys are exposed on the frontend. The app is built with Vite for fast development and optimized builds.
