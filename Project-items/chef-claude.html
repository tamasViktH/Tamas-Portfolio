Chef Claudeweb app

HTML & CSS

GitHub Repo Live version

Chef Claude is a recipe-generating AI assistant built using React, with real-time integration of Anthropic's Claude 3 and Hugging Face’s Mixtral models. Designed as part of the Scrimba Frontend Developer Career Path, this project helps users find meal ideas by entering ingredients they already have. The app features a sleek, user-friendly UI and delivers AI-generated recipe instructions in Markdown-rendered format.

Users can input one or more ingredients, add them to a list, and generate a recipe suggestion tailored to those items. I implemented this functionality using modern React patterns including state hooks, controlled components, and asynchronous API requests. The app communicates securely with AI models through environment variables (never exposing keys on the frontend), and is bundled using Vite for fast development and builds.
