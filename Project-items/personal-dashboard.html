Personal Dashboard Productivity Tool

HTML5 | CSS3 | JavaScript (ES6+)

GitHub Repo Live version

The Personal Dashboard is a minimalist, real-time homepage designed to deliver useful daily information at a glance — including the current time, weather conditions, and live cryptocurrency prices — all layered over a dynamic background.

Built as part of the Scrimba Frontend Developer Career Path, this project emphasizes real-world problem solving using asynchronous JavaScript, API integration, and responsive layout design. I worked with the Unsplash API for rotating background images, CoinGecko for crypto data, and OpenWeatherMap to fetch location-based weather updates.

Throughout the build, I used fetch, async/await, and try/catch to gracefully handle data fetching and errors. The project strengthened my skills in async programming, DOM manipulation, and building API-driven user interfaces with clean, semantic HTML and custom CSS.
