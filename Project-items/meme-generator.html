Meme Generator React App

React | JavaScript (ES6+) | CSS3 | Meme API

GitHub Repo Live version

Meme Generator is a dynamic React app that lets users generate custom memes by entering their own top and bottom text. It pulls from a collection of meme templates and overlays the text in real time, delivering a fun and responsive user experience.

The app was built using React and practices core concepts like controlled components, state management with useState, and event handling. Each meme image is selected randomly from an API response and updated on button click, while user input is reflected instantly on the rendered meme.

This project helped me strengthen my understanding of form handling, component composition, and dynamic DOM updates in React. It also taught me how to work with external APIs, manage application state cleanly, and build interactive UI elements that respond to user input.
