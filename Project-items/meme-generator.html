GitHub Repo Live version

The Meme Generator is a playful web app I created during Scrimba’s Frontend Developer Career Path to practice building interactive React applications. It allows users to enter top and bottom text and overlay it on randomly selected meme images. This simple but fun tool highlights how powerful and quick it is to build dynamic UIs with React.

The app uses a public meme image API to fetch a wide variety of meme templates. I implemented state management using React’s `useState` hook, and created a responsive, styled layout using custom CSS. The form dynamically updates the meme preview as users type, demonstrating controlled inputs and reactivity.

This project helped me get hands-on experience with React fundamentals such as functional components, props, hooks, and conditional rendering. It also gave me the chance to polish my CSS skills, particularly with layout and text styling over images. Overall, it's a great example of how coding can be both educational and entertaining.
